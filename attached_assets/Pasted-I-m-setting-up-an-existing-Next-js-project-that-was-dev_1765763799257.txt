### To Get Running

1. Run `npm install` to install dependencies
2. Set up the environment variables in Secrets
3. Run `npm run dev` to start the development server
4. The app should be available on port 5000 (mapped to external port 80)

### Important Notes

- **DO NOT** modify the `.replit`, `replit.nix`, or `next.config.js` files - they're already configured
- **DO NOT** reference the `_replit_backup/` folder - it contains legacy code from before the Next.js migration
- All routes are in `src/app/`
- All components are in `src/components/`
- Product data is in `src/data/catalog.ts`
- The project uses App Router, NOT Pages Router

### If You See Errors

1. **Module not found**: Run `npm install`
2. **Port in use**: Run `pkill -f next` then `npm run dev`
3. **Database errors**: Check that `DATABASE_URL` is set correctly in Secrets
4. **Build errors**: Run `npm run build` to see detailed TypeScript errors
